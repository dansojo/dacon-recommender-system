## ✔️2.데이터
  1) view_log.csv : 유저가 기사를 조회한 로그 데이터
  2) article_info.csv : 기사에 대한 정보 데이터
  3) sample_submission.csv : 제출 양식, 한 유저에게 5개의 기사를 추천한 리스트
     
🙌 데이터에 대한 자세한 설명은 data 폴더 readme를 확인해 주세요.
